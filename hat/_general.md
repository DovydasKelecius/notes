### OSI Model

1. P Physical - data cables, CAT6
2. D Data - Switching, MAC addresses
3. N  Network - IP addresses, Routing
4. T Transport - TCP/UDP
5. S Session - Session management
6. P Presentation - WMV, JPEG, MOV
7. A Application - HTTP, SMTP
